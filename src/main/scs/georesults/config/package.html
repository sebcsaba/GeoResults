Ez a csomag egy segédosztályon kívül csak konfigurációs állományokat tartalmaz. Ezek a kövekezők:
  • sablonok.config: ez tartalmazza az új verseny létrehozásakor felkínált sablonokat. A fájl felépítése a szokásos properties típusú fájloknak felel meg: egy sorban a név egy sablont tartalmazó fájl neve, az érték pedig a hozzá megjelenítendő név. Ha ez a név @ karakterrel kezdődik, akkor a név többi részét a globális szótár egy kulcsának tekinti a program.
  • *.xml: Az új verseny létrehozásakor használt sablonok, GeoResults2006 XML formátumban. Ez a formátum a scs/javax/rdb/rdbxml/rdbxmlmapping.dtd által megadott módon írja le a {@link scs.georesults.om} csomag osztályainak példányait.
  • install.sql: a program telepítéséhez szükséges SQL utasításokat tartalmazó fájl.
Felhívnám a figyelmet arra, hogy a program konfigurációjakor létrejön egy database.config fájl, amely az adatbázishoz kapcsolódás paramétereit tartalmazza a properties fájlok formátumában. Ez a fájl a scs.georesults.config csomag könyvtárában jön létre, a webalkalmazás WEB-INF/classes könyvtárán belül.
