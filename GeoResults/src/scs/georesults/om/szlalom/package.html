A szlalomversenyek eredményeit (futamjait) tartalmazó osztályok.

A fájlok hármas csoportokat alkotnak, amely csoport a modell egy osztályát reprezentálja:

  • az adatmodell egy osztály-tábla leképezését leíró XML fájl
  • az XML alapján generált alaposztály
  • az alaposztályból származó implementációs osztály, a neve végén Impl postfixszel
