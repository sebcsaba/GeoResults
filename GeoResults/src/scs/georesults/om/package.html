A program adatmodell-rétegét tartalmazó csomag. A program perzisztens osztályainak struktúrája alapján nyolc csomagból áll. Ezek mindegyikében a fájlok hármas csoportokat alkotnak:
  • az adatmodell egy osztály-tábla leképezését leíró XML fájl
  • az XML alapján generált alaposztály
  • az alaposztályból származó implementációs osztály, a neve végén Impl postfixszel
